# Upstream pin for vm/d-modem/

# This directory is a vendored snapshot of the d-modem.c glue file from
# the D-Modem project. d-modem.c is the PJSIP media-port subclass that
# bridges a PJSIP call to slmodemd's socketpair; it is the integration
# layer the slmodemd-pjsip backend relies on.
#
# We do NOT fork d-modem.c: when upstream releases changes, re-run
# scripts/fetch-d-modem.sh (or copy d-modem.c verbatim from a fresh
# clone), update the commit hash and file SHA256 below, and rebuild.
#
# The d-modem.c source is GPLv2-or-later; see licenses/D-MODEM-NOTICE
# and the copyright header inside d-modem.c itself.
#
# See IMPLEMENTATION.md and COPYING for the policy.
